# Modern Todo List

A modern and responsive **Todo List Web Application** built using **HTML, CSS, and JavaScript**.  
This project helps users manage daily tasks efficiently with features like editing tasks, marking tasks as complete, clearing completed tasks, and a dark mode for better user experience.

---

## 🚀 Features

- ➕ Add new tasks quickly
- ✅ Mark tasks as completed
- ✏️ Edit existing tasks
- 🗑️ Delete tasks individually
- 🧹 Clear all completed tasks with one click
- 📊 Dynamic task counter showing remaining tasks
- 🌙 Dark mode toggle
- 💾 Persistent storage using **browser localStorage**
- 📱 Responsive and modern UI

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

## ⚙️ How It Works

1. Enter a task in the input field.
2. Click **Add** or press **Enter** to add the task.
3. Click the **check button** to mark a task as completed.
4. Click **Edit** to update a task.
5. Click **Delete** to remove a task.
6. Click **Clear Completed** to remove all completed tasks.
7. Use the **Dark Mode button** to switch themes.

All tasks are automatically saved using **localStorage**, so they remain even after refreshing the page.
