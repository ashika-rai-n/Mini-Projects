# 🎨 Color Palette Generator

A sleek and interactive web app to generate beautiful random color palettes with a single click. Designed with a modern UI and smooth user experience.

---

## 🚀 Features

* 🎲 Generate 5 random color palettes instantly
* 📋 Copy any color code to clipboard with one click
* 🌈 Dynamic background gradient based on generated palette
* 🎯 Click on any color to set it as background
* ⚡ Smooth animations and responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid + Gradients)
* JavaScript (DOM Manipulation + Clipboard API)

3. Double-click `index.html`

OR

Open in VS Code and use Live Server extension

## 💡 How It Works

* Generates random hex color codes
* Updates UI dynamically using JavaScript
* Applies colors to palette and background
* Uses event delegation for efficient click handling
