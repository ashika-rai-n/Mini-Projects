# 🧮 Scientific Calculator

A sleek, dark-themed scientific calculator built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — runs entirely in the browser.

## ✨ Features

- **Basic Operations** — Addition, subtraction, multiplication, division
- **Scientific Functions** — sin, cos, tan, log, ln, √, xʸ, |x|, %
- **Constants** — π (pi) and e (Euler's number)
- **Degree / Radian toggle** for trigonometric functions
- **Memory functions** — MS (Memory Store) and MR (Memory Recall)
- **Live preview** — See result update as you type
- **Auto parenthesis closing** on `=`
- **Keyboard support** — Type expressions directly
- **Backspace & AC** — Delete last character or clear all

## 🚀 How to Run

Just open `index.html` in any modern browser. No setup required!

```bash
# Clone the repo and open
open index.html
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Number input |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last |
| `Escape` | Clear all |
| `( )` | Parentheses |
| `^` | Power |
| `%` | Modulo |
