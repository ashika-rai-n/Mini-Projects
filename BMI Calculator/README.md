# BMI Calculator

A simple **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.

## Features
- Takes weight (kg) and height (cm) as input  
- Calculates BMI value  
- Displays BMI category (Underweight, Normal, Overweight, Obese)  
- Supports **Dark/Light Mode Toggle** for better user experience  

## How to Use
1. Download all three files (index.html, style.css and script.js).
2. Place them in the same directory.
3. Open `index.html` in your browser.  
4. Enter your weight and height.  
5. Click **Calculate BMI** to see your BMI and health category.  
6. Use the toggle button to switch between dark and light mode.
