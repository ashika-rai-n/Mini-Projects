# Expense Tracker App

A simple and user-friendly Expense Tracker built using HTML, CSS, and JavaScript.  
It helps users manage their income and expenses efficiently.

---

## Features

- Add income and expense transactions  
- Delete transactions  
- Automatic balance calculation  
- Shows total income and expenses  
- Data stored using Local Storage 
- Clean and responsive UI  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## How to Run

1. Clone the repository:
   git clone https://github.com/Aditi-Sachdeva/Mini-Projects.git

   cd Mini-Projects

   Open `index.html` in your browser

## Usage

- Enter description and amount  
- Positive = Income  
- Negative = Expense  
- Click Add Transaction  
- Delete using Remove button
