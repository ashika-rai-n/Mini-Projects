# 🗓 Daily Focus Planner

A simple and elegant productivity tool that helps you organize and track your daily tasks.  
Plan your day, stay focused, and monitor your progress with a clean and responsive interface.

---

## 🚀 Features

- Add daily tasks
- Optional task time with **AM/PM format**
- Mark tasks as **Done / Undo**
- Delete individual tasks
- Dynamic progress bar
- Persistent storage using **localStorage**
- Modern glassmorphism UI
- Responsive design

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Browser LocalStorage**

No frameworks or backend required.

## 💡 How It Works

- Tasks are stored in **browser localStorage**
- Each task contains:
  - Task text
  - Optional time
  - Completion status
- The **progress bar updates automatically** based on completed tasks.
